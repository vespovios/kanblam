# KanBlam documentation

KanBlam is a personal and small-team task manager. Five ways to look at the
same work — Kanban, Eisenhower, Calendar, DayDash, Tasks list — backed by
one source of truth.

The screenshots throughout these docs come from **Stratos-1**, a demo
workspace planning a high-altitude balloon launch — four projects,
thirty-odd tasks, one deadline that doesn't move. If you can run a
balloon mission on it, your week will be fine.

## Got stuck?

This is a young product still finding its sharp edges. If something
doesn't work the way the docs say it should, the docs are wrong — file
an issue on [GitHub](https://github.com/vespovios/kanblam/issues).
